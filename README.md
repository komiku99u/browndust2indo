# BrownDust2 Indonesia Guide

Website panduan dan referensi pemain BrownDust2 berbahasa Indonesia.

🌐 Website: https://bd2.pages.dev/

## Fitur

- 📖 Panduan Pemula
- ⚔️ Panduan Gear
- 🏆 Tier List Terbaru
- ❓ FAQ (Pertanyaan yang Sering Ditanyakan)
- 👥 Komunitas BrownDust2 Indonesia
- 🔞 Fated Guest Interactions
- 🔞 Prestige Skin Interactions

## Struktur Konten

| Menu | Deskripsi |
|--------|-----------|
| Komunitas | Link komunitas dan grup pemain |
| Panduan Pemula | Dasar-dasar bermain BrownDust2 |
| Panduan Gear | Penjelasan sistem gear dan rekomendasi |
| Tier List | Ranking karakter terbaru |
| FAQ | Pertanyaan yang sering ditanyakan pemain |
| Fated Guest Interactions | Panduan interaksi Fated Guest |
| Prestige Skin Interactions | Panduan interaksi Prestige Skin |

## Teknologi

- HTML5
- CSS3
- Material Icons
- Font Awesome

## Tujuan Proyek

Website ini dibuat untuk membantu pemain BrownDust2 Indonesia mendapatkan informasi yang mudah diakses mengenai gameplay, karakter, gear, dan berbagai panduan lainnya.

## Kontribusi

Jika menemukan informasi yang kurang tepat atau ingin menambahkan panduan baru, silakan membuat Issue atau Pull Request.
